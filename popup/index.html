Messenger for poweruser

Shortcuts:

  • i : focus chat input
  • / : focus search input
  • Escape : blur all
  • j : scroll bottom *
  • k : scroll top *
  • Tab : switch between chat and sidebar *
  • h or l : switch between chat and sidebar *
  • o : open information sidebar *
  • u : send a like or primary emoji *
  • a : open active contacts *
  • d : switch dark mode on/off *
    • * (when no input are focused)
